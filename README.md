# Socialmedia_Deepfake_detection-
“AI-powered Deepfake Detection System using EfficientNetB0 and LSTM for identifying manipulated facial videos on social media.”

Deepfake Detection System

This project is an AI-powered Deepfake Detection System designed to identify manipulated or synthetic videos by analyzing facial patterns and temporal inconsistencies. The system uses EfficientNetB0 for deep feature extraction from cropped facial frames and LSTM (Long Short-Term Memory) networks to capture sequential frame patterns for accurate deepfake classification.

The project focuses on detecting deepfake content commonly shared on social media platforms by leveraging computer vision and deep learning techniques.

Features
Face extraction and preprocessing from videos
Deep feature extraction using EfficientNetB0
Temporal sequence learning using LSTM
Real vs Fake video classification
User-friendly interface for video upload and detection
Deep learning-based prediction with improved accuracy
Technologies Used
Python
PyTorch
OpenCV
EfficientNetB0
LSTM
NumPy
Flask/Frontend (if used in your project)
Objective

To build an intelligent system that helps detect manipulated media content and reduce the spread of misinformation caused by deepfake videos on social media platforms.

Applications
Social media content verification
Fake media detection
Cybersecurity and digital forensics
News and media authenticity verification
